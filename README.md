# graphql-federation-go

Graphql Federation Implementation Example with Go

## Try it
### Clone project
```bash
git clone https://github.com/aeramu/graphql-federation-go.git
```
### Run sales-service
```bash
cd ./sales-service
go run .
```
### Run order-service
```bash
cd ./order-service
go run .
```
### Run gateway
```bash
cd ./gateway
yarn
yarn start
```
